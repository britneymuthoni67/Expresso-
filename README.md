# 🎓 Graduation Surprise Website

A fun and creative graduation announcement website with:
- Floating hearts ❤️
- Popup invitation 🎉
- Glowing graduation card ✨
- Moving super car 🚗
- Horror flashing effects ⚡
- Ghostface jump scare 👻

## 📸 Features
- Custom graduation photo
- Animated effects
- Horror + celebration theme
- Responsive design

## 🛠 Technologies Used
- HTML
- CSS

## 🚀 How To Run
1. Download the project
2. Open the folder in VS Code
3. Add your images:
   - me.jpg
   - car.png
   - ghostface.png
4. Open `index.html`
5. Run with Live Server
